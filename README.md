# Kung Fu Chess

## משחק מקומי לשני שחקנים

התקינו פעם אחת את התלויות:

`python -m pip install -r requirements.txt`

לאחר מכן לחצו פעמיים על `start_local_match.bat` (או הריצו
`python start_local_match.py`). ייפתחו שרת מקומי ושני חלונות משחק:
השחקן הראשון הוא לבן והשני שחור. לחיצה שמאלית בוחרת כלי ויעד; לחיצה
ימנית מפעילה Jump. יוצאים מכל חלון עם `Q` או `Esc`.
# kung-fu-chess
Python server implementation of a real-time Kung-Fu Chess game, developed with a focus on clean architecture, object-oriented design, extensibility, and scalable multiplayer support.
